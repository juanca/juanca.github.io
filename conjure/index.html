Built for spatial computing

A voice-first, multi-agent interface designed from the ground up for Vision Pro.

🎤

Voice-first

Hold to talk. Your speech is transcribed and sent to the agent. Queue messages while the agent is busy.

◆

Multi-agent

Spawn as many agents as you need. Each runs in its own window — arrange them anywhere in your space.

⚡

Live streaming

Watch responses appear in real-time. See tool use, file reads, bash commands, and code generation as they happen.

☁

Cloud mode

Call the Anthropic API directly from Vision Pro. No Mac needed — just paste your API key and go.

💻

Mac mode

Connect to your Mac via the Conjure Bridge for full Claude Code access — read files, write code, run commands.

🔒

Private & secure

API keys in Keychain. TLS encryption. PIN-based pairing. No telemetry. Everything stays on your devices.

Two ways to use

Cloud

Direct API

Talk to Claude directly via the Anthropic API. Great for questions, brainstorming, and general assistance.

  1. Open Conjure on Vision Pro
  2. Paste your Anthropic API key
  3. Spawn an agent and start talking
Mac

Conjure Bridge

Connect to your Mac for full Claude Code power — file access, code generation, terminal commands.

  1. Install Conjure Bridge on your Mac
  2. Run the bridge server
  3. AirDrop the CA certificate to Vision Pro
  4. Install and trust the certificate
  5. Enter the PIN from the Bridge menu bar
